# QAntum QA Tool

> AI-powered CLI for test automation

## Installation

```bash
npm install -g @QAntum/qa-tool
```

## Usage

```bash
# Initialize project
QAntum init

# Run tests
QAntum test

# Scan site and generate tests
QAntum scan https://example.com

# Generate report
QAntum report --open

# Heal broken selectors
QAntum heal --apply
```
